# Notebook LM – Informatica, SER, TEPI

Sito statico con i notebook LM organizzati per **anno → classe → lezione**.

Contiene tre file:
- `index.html` – pagina principale (Vue 3 via CDN)
- `style.css` – stile del sito
- `lm-data.json` – dati delle lezioni

Pubblicato tramite **GitHub Pages** dal branch `main`.

Per aggiornare il sito basta modificare `lm-data.json` e fare commit.
